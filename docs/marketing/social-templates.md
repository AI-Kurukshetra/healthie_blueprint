## Twitter/X Thread (detailed)

1/ Just launched CareSync on @ProductHunt 🏥

A modern EHR platform for virtual clinics — built in 10 hours during a hackathon. Here's what I built and how 🧵

2/ The problem: Legacy EHR systems were built for hospitals. Virtual clinics need something simpler, faster, and modern.

CareSync gives providers, patients, and admins exactly the tools they need — nothing more.

3/ For Providers:
→ Dashboard with today's appointments and patient stats
→ Patient records with full medical history
→ SOAP clinical notes linked to every visit
→ Schedule management

4/ For Patients:
→ Book appointments with available providers
→ View health records (conditions, meds, allergies)
→ Message their care team
→ Clean, friendly interface

5/ Security is built into the foundation:
→ Row Level Security on every database table
→ Patients only see their own data
→ Providers only access their assigned patients
→ Three-role access control (patient, provider, admin)

6/ Tech stack:
→ Next.js 15 (App Router + TypeScript)
→ Supabase (Auth + PostgreSQL + RLS)
→ shadcn/ui + Tailwind CSS
→ Deployed on Vercel
→ Built with Codex CLI
