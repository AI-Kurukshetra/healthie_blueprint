## Product Name
CareSync

## Tagline (60 chars max)

## Short Description
CareSync is an API-first virtual health platform that helps providers manage patients, schedule appointments, and write clinical documentation — while patients book visits, view their records, and communicate with their care team. Built as a clean, modern alternative to legacy EHR systems like Healthie.

## Maker Comment (post as first comment on launch)

"Hey Product Hunt! 👋

I built CareSync during a 10-hour hackathon — AI Mahakurukshetra by Bacancy.

The problem: Virtual clinics need EHR tools, but existing options are either built for hospitals (too complex) or come with steep learning curves and pricing. CareSync strips it down to what matters — scheduling, clinical notes, patient records, and role-based access.

Built with Next.js 15, Supabase (with Row Level Security on every table), and deployed on Vercel. Used Codex CLI for AI-assisted development.

Key features:
→ Three-role system (patient, provider, admin)
→ SOAP clinical notes linked to appointments
→ Patient booking flow
→ Role-based dashboards
→ Database-level security

Would love your feedback — what features would you want to see next?"
